## Bugs Found

- [ ] BUG-p13.6-1: Compile error on function pointer declaration. The instrumenter emits `__crow_decl("(*fp)(int, int)", &(*fp)(int, int), sizeof((*fp)(int, int)), "int", 4)` which is invalid C -- it tries to take the address of `(*fp)(int, int)` (a function call expression) rather than `&fp`. The instrumenter does not handle function pointer declarator syntax; it should emit `__crow_decl("fp", &fp, sizeof(fp), "int(*)(int,int)", 4)` or similar. Zero steps and zero snapshots are produced.
